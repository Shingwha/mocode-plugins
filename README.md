# Mocode Plugins

My personal plugin collection for [mocode](https://github.com/Shingwha/mocode).

## Plugins

### better-write

Enhanced write tool with logging and backup support.

### test-plugin

A test plugin for development.

## Installation

```bash
/plugin install <github-url>
```

## Plugin Structure

Each plugin follows this structure:

```
plugin-name/
├── plugin.py       # Required: Plugin implementation
└── plugin.yaml     # Optional: Metadata
```
